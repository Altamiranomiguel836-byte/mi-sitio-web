Sobre mí

Diseñador y desarrollador web independiente detrás de Ale's Design.

Miguel Gálvez

Fundador · Ale's Design

Soy desarrollador web con base en Alajuela, Costa Rica. Creé Ale's Design para ayudar a negocios y emprendedores a tener una presencia digital profesional: sitios que cargan rápido, se ven bien en cualquier dispositivo y comunican con claridad lo que ofreces.

Me apasionan la tecnología, la programación y el diseño de interfaces. Cada proyecto es una oportunidad de aprender, probar ideas nuevas y entregar algo del que puedas sentirte orgulloso.

HTML & CSS JavaScript UI / UX Responsive Apps web E-commerce SEO básico
